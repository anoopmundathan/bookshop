# bookshop
React Redux Bookshop App 

## Getting Started
### Setup
```
$ git clone https://github.com/anoopmundathan/bookshop.git
$ cd bookshop
```
### Building
Install dependencies:
```
$ npm install
```
#### npm scripts
```$ npm run dev``` - Run node server - webpack watch mode

```$ npm start``` - Run node server - webpack production mode
